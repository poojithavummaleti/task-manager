# Task Manager (To-Do App)

 Description
This is a simple To-Do List web application built using HTML, CSS, and JavaScript. It allows users to add and delete tasks easily.

# Features
- Add new tasks
- Delete tasks
- Input validation (prevents empty tasks)
- Clean and simple user interface

- HTML

- JavaScript

 How It Works
1. Enter a task in the input box
2. Click on the **Addtask  button
3. The task will appear in the list
4. Click **Delete** to remove the task
5. Project Structure
- index.html → Structure of the app
- style.css → Styling of the app
- script.js → Functionality (add/delete tasks)
